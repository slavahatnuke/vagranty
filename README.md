# Vagranty

## Start
* `mkdir vm && cd vm`
* `curl -sSL https://raw.github.com/slavahatnuke/vagranty/master/install.sh | sh`
* `vagrant up`

## Settings
* `Vagrantfile.project.yml` VM configuration

## Inside
* `vagrant ssh`
* `cd /project`
* `ls`

## Provision
* `cd /vagrant`
* `vitality provision.yml`
